# react-start-kit
react脚手架

用于快速创建基于 React + Webpack 为技术栈的前端项目。

# 特性
- 支持JSX语法，ES6
- Sass预处理器
- 编译完成自动打开浏览器
- 支持热更新，自动刷新
- 区分开发环境和生产环境
- 支持HTML模版、图片、图标字体等资源的编译


# 技术栈
- react
- react-router
- webpack
- babel
- redux
- redux-actions
- redux-promise
- redux-saga
- redux-thunk
- redux-logger
- lodash
- isomorphic-fetch
- classnames
- normalize.css

# Usage
- 安装：npm install
- 开发：npm start
- 生产：npm run build
